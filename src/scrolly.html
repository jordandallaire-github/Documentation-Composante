Scrolly

La composante Scrolly est un outil permettant de créer des animations d'apparition ou de disparition d'éléments lorsqu'ils deviennent visibles dans la fenêtre du navigateur.

Comment l'utiliser

Pour utiliser cette composante, il faut ajouter le data-component Scrolly sur le body. Ensuite, il faut ajouter des éléments avec l'attribut "data-scrolly" pour indiquer quel type d'animation doit être appliqué à l'élément.

Dans la structure TimTools, la ComponentFactory s'occupe de l'instantiation des composantes.

Voir la componentFactory

Les animations

Pour les animations, c'est à vous de determiner le type d'animation que vous voulez. Les exemples si-dessous sont pour vous montrer des exemples d'animations que vous pouriez utiliser ou en faire une nouvelle, ceux-ci sont fromLeft, fromBottom, fromRight, fromTop, scale, rotation.

À notez que la classe .has js ajouter par le script, est nécessaire pour englober tout le code scss de la composante pour assurez le fonctionnement de votre site même si le code js n'embarque pas

Votre propre animation

Pour ajouter vos propre animation il faut ajouter dans le scss le nom data-scrolly avec le nom de votre animation et l'ajouter dans le html. Ensuite vous pouvez vous amusez :)

Paramètre

Le paramètre noRepeat permet de spécifier que l'animation ne doit être déclenchée qu'une seule fois. Cela permet d'éviter de répéter l'animation plusieurs fois et d'améliorer les performances du site. Pour cela il faut rajouter le data-no-repeat à l’endroit souhaité.

Voir un démo du paramètre noRepeat
